## Project Overview
This project applies principles of complexity science to model modern IT infrastructure. It simulates a distributed cloud computing network as a complex dynamical system to observe **Self-Organized Criticality (SOC)**. 

Using a two-dimensional Bak-Tang-Wiesenfeld (BTW) sandpile model, the simulation demonstrates how localized computational task overloads (noise) propagate through network topologies via load-shedding protocols, triggering cascading failures (avalanches). The project proves that without centralized tuning, the network naturally evolves into a critical state where catastrophic failures follow a mathematically unavoidable power-law distribution.

## Repository Structure
* `simulation.py`: The main Python script containing the BTW sandpile network simulation.
* `avalanche_distribution.png`: The log-log plot generated by the simulation, illustrating the power-law distribution of avalanche sizes.
